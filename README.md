# Country Finder API Service

## Setup Instructions

1. **Download the project**
   - Make sure you have Java/JDK installed
   - Ensure you have an active internet connection

2. **Run the project**
   - Navigate to `CountrydestinationapiApplication.java` class
   - Right-click and select **Run using Spring boot**

3. **Access the API**
   - Open Chrome browser
   - Paste the following URL: `http://localhost:8080/swagger-ui/index.html#/`

4. **Use the API**
   - Enter your input (3 characters required for both source and destination)

5. **View Results**
   - The API will return the results based on your input
