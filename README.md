Country finder api service.

1.) Download the project.
  a. Make sure you have Java/JDK/Active Internet Connection
2.) Run the project by right clicking on CountrydestinationapiApplication.java class and select option Run using Spring boot
3.) Head to chrome and paste the following http://localhost:8080/swagger-ui/index.html#/
4.) Type in your input which consists of 3 characters for both source and destination
5.) Result!
